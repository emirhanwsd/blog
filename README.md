## Blog

Source codes of my personal blog

# Installation

```sh
npm install

npm run dev
```

# Technologies

- [React](https://reactjs.org/)
- [NextJS](https://nextjs.org/)
- [TailwindCSS](https://tailwindcss.com/)

# Features

- [x] List blogs
- [x] View blogs
- [x] Blog not found page
- [x] Add new blog easily
- [x] Responsive design
- [ ] Markdown support
- [ ] Language support
